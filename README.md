# Новостной сайт

## Описание проекта
Это веб-приложение для публикации и обсуждения новостей, разработанное по архитектуре **MVC на PHP**.  
Пользователь может:

- Просматривать новости с текстом и изображениями.
- Читать новости по категориям.
- Оставлять комментарии под новостями.
- Регистрироваться и авторизовываться на сайте.

Администратор может управлять новостями и комментариями через админ-панель.

---

## Особенности

### Новости
- Отображение последних новостей на главной странице.
- Просмотр всех новостей или новостей по категории.
- Страница отдельной новости с комментариями.

### Комментарии
- Добавление комментариев зарегистрированными пользователями.
- Публичное отображение комментариев.

### Админ-панель
- Добавление новостей с изображениями.
- Редактирование и удаление новостей.
- Управление комментариями.

### Пользовательский интерфейс
- Простая навигация по категориям.
- Формы регистрации и входа.
- Отдельный интерфейс для администратора.

---

## Используемые технологии
- PHP (MVC)
- MySQL
- HTML5, CSS3

---

## Инструкция по запуску
1. Скачайте проект или клонируйте репозиторий.
2. Настройте базу данных MySQL и импортируйте файл структуры базы.
3. Настройте подключение к базе данных в файле конфигурации (`config.php`).
4. Запустите локальный сервер (например, XAMPP, MAMP, Laragon) и откройте:
http://localhost/папка_проекта/
